# Fontes do Manual de Marca Excellus

Adicione os arquivos de fonte nesta pasta:

## CoFo Gothic VF (principal)
- `CoFoGothic-VF.woff2` ou `CoFoGothic-VF.woff`
- `CoFoGothic-VF-Italic.woff2` (opcional, para itálico)

## Abril Display (secundária)
- `AbrilDisplay-Regular.woff2` ou `AbrilDisplay-Regular.woff` ou `.ttf`
- `AbrilDisplay-Italic.woff2` (opcional, para destaque elegante)

Sem os arquivos, o site usará as fontes de fallback (system-ui e Georgia).
